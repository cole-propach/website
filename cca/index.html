Cole's Cellular Automaton

Controls

Left Click: place/erase cell
E: swap between placing and erasing
Right Arrow: progress to the next generation (hold to fast forward)
C: load random chaos
Number Keys: load presets from the list below

1: Cycle with 81 states
2: Autofilling Grid
3: Fenced Chaos
4: Rocket
5: Garden
6: Snowflake Explosion
7: Snowflake
8: Flower
9: Flower Explosion
